Hello, world!

Bem-vindo ao seu novo aplicativo de single-page, construído com:

  • ASP.NET Core e C# para código do servidor multiplataforma
  • Angular e TypeScript para código do lado do cliente
  • Bootstrap para layout e estilo das páginas

Para ajudar você a começar, também configuramos:

  • Integração com CLI angular . No modo de desenvolvimento, não há necessidade de executar ng serve . Ele é executado em segundo plano automaticamente, para que seus recursos do cliente sejam criados dinamicamente sob demanda e a página seja atualizada quando você modificar qualquer arquivo.
  • Construções de produção eficientes . No modo de produção, os recursos em tempo de desenvolvimento são desativados e sua configuração de dotnet publish invoca automaticamente ng build para produzir arquivos JavaScript compilados e minificados antecipadamente.

O subdiretório ClientApp é um aplicativo CLI Angular padrão. Se você abrir um prompt de comando nesse diretório, poderá executar qualquer comando ng (por exemplo, ng test ) ou usar npm para instalar mais pacotes nele.
